# API reference

This folder holds **API reference** content for AI Fabrix. It is synced to the Document360 **`api-docs`** project version (see `README.md` and `npm run sync-document360:api-docs`).

User-facing guides and platform overview remain under **`docs/`**, which also powers the Jekyll site.

Add pages here as `.md` files with sibling `.yaml` metadata (same pattern as `docs/`).
